# Student Management System - Java

## Overview
This is a **console-based Java mini project** for managing student information.  
It demonstrates **CRUD operations (Create, Read, Update, Delete)** using **file handling** in Java.

---

## Features
- Add new student records
- View all students
- Search for a student by ID
- Update student information
- Delete a student
- Data is stored in a text file (`students.txt`)

---

## Technologies Used
- Java SE (JDK 17)
- File Handling (`java.io`)
- Collections (`ArrayList`)
- Eclipse IDE

---

## Project Structure
Student-Management-System-Java/
├── src/
│ └── mini_project/
│ └── StudentManagementSystem.java
└── data/
└── students.txt

## How to Run
1. Import the project in **Eclipse IDE**  
2. Run `StudentManagementSystem.java`  
3. Follow the console menu to perform operations
